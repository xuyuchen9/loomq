# LoomQ v0.6.x Roadmap

## v0.6.0 (2026-04-10) - Netty HTTP 层重构

### 已完成

- [x] Netty 原生 HTTP 服务器
- [x] RadixRouter 前缀树路由器
- [x] I/O 线程与虚拟线程分离
- [x] Intent API 端点迁移
- [x] 性能达标 (414K QPS)
- [x] V5 集成测试通过

### 性能提升

| 指标 | v0.5.x | v0.6.0 | 提升 |
|:---|:---|:---|:---|
| POST /v1/intents QPS | ~8K | ~414K | **50x** |
| Health QPS | ~14K | ~5.6M | **400x** |
